About files

There are many types of files that can be stored on a computer, such as text, image, audio, video, and executable files. Each file type has a specific format that defines how the data is organized and interpreted by the computer. The file type is usually indicated by the file extension, which is the part of the file name after the dot. For example, a file named `document.docx` is a Microsoft Word document file, and a file named `picture.jpg` is a JPEG image file.

Some files can have different versions, which means that they have been modified or updated over time. File versioning is a feature that allows you to keep track of the changes made to a file and restore a previous version if needed. File versioning is available in some applications, such as Microsoft 365 or Google Drive, and some backup systems, such as Time Machine for Mac or File History for Windows¹².

Another aspect of files is the bitness, which refers to the amount of memory that a file or a program can use. There are two common bitness options: 32-bit and 64-bit. A 32-bit file or program can use up to 4 GB of memory, while a 64-bit file or program can use more than that. The bitness of a file or a program depends on the operating system and the processor of the computer. For example, a 64-bit file or program can only run on a 64-bit operating system and a 64-bit processor³.
