Image Formats

JPEG – (or JPG) Joint Photographic Experts Group: This is a lossy image format, which means it compresses the image data and reduces the file size, but also lowers the image quality.

GIF – Graphics Interchange Format: This is a lossless image format, which means it preserves the image data and quality, but has a larger file size.

PNG – Portable Network Graphics: This is also a lossless image format, but it supports more colors and transparency than GIF.

WebP – Web Picture Format: This is a newer image format that can be either lossy or lossless, depending on the compression level.

HEIF (or HEIC) – High Efficiency Image File Format: This is a lossy image format that uses advanced compression techniques to reduce the file size and maintain the image quality.

SVG – Scalable Vector Graphics: This is an XML-based vector image format that can store and display graphics, logos, and illustrations that are based on mathematical equations, rather than pixels.
