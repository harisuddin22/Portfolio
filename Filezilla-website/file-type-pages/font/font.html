Font Formats

TrueType Font (TTF). This is a cross-platform font format that can store both display and printer data in a single file.

OpenType Font (OTF). This is an advanced font format that can support thousands of glyphs, multiple languages, and advanced typographic features.

Web Open Font Format (WOFF). This is a web font format that can compress font data to reduce file size and loading time.

Embedded OpenType Font (EOT). This is another web font format that can embed font data into web pages for compatibility with older browsers.

Scalable Vector Graphics Font (SVG). This is a vector-based font format that can create fonts with high-quality graphics and animations.
