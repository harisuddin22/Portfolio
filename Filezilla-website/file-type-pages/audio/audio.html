Audio Formats

OGG: A compressed lossy audio format that is free and open-source.

WAV: An uncompressed lossless audio format developed by IBM and Microsoft.

MP3: A popular compressed lossy audio format that reduces the file size significantly, but also loses some sound quality.

FLAC: A lossless compression format that supports high-resolution sample rates, and stores metadata. It takes up about half the space of WAV, and is royalty-free.

M4A: A compressed lossy audio format that is similar to AAC, but has better sound quality. It is used for Apple Music streaming and iTunes downloads.

AIFF: AIFF is an uncompressed lossless audio format that was developed by Apple. AIFF files are similar to WAV files, but use a different way of storing the audio data. AIFF files have high sound quality, but also large file sizes.
