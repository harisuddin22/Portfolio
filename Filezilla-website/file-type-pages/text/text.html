File Types

Text Formats

DOC : A document file format used by Microsoft Word. It can contain text, images, tables, and other elements. It is editable and compatible with most word processors¹.

PDF: A portable document format that preserves the layout and appearance of any file. It can be viewed with Adobe Acrobat or other PDF readers. It is not easily editable, but it is secure and widely used².

HTML: A hypertext markup language that defines the structure and content of a web page. It can be opened and edited with a text editor or a web browser. It uses tags and attributes to format text, images, links, and other elements¹.

XML: An extensible markup language that stores and transports data. It can be used to create custom tags and schemas for different purposes. It is readable by both humans and machines. It can be validated and transformed with other languages such as XSLT and XPath¹.

TXT: A plain text file that contains only text without any formatting. It can be opened and edited with any text editor or word processor. It is simple and universal, but it does not support images, tables, or other elements³.

eBook: This is a format that is designed for electronic books and digital publications. It can include text, images, audio, video, and interactive features.
