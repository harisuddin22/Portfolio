File Types

Executable Formats

EXE. This is the standard executable file format for Windows operating systems. It can contain machine code, resources, and metadata for running a program.

ELF. This is the standard executable file format for Unix and Unix-like operating systems, such as Linux, FreeBSD, and Solaris. It can store executable code, shared libraries, and core dumps.

APP. This is the standard executable file format for macOS operating systems. It is actually a folder that contains the executable code, resources, and metadata for running an application.

APK. This is the standard executable file format for Android operating systems. It is a compressed archive that contains the executable code, resources, and metadata for running an application on a mobile device.

COM. This is an old executable file format for DOS and Windows operating systems. It can contain only 64 KB of machine code and no other information.

CGI. This is a special executable file format for web servers. It can contain a script or a program that generates dynamic web content.
