Spreadsheet Formats

Microsoft Excel. This is one of the most widely used spreadsheet formats, which can store data in rows and columns, perform calculations, create charts, and apply various formatting options.

Google Sheets. This is an online and free spreadsheet format, which allows users to create, edit, and share spreadsheets on the web. Google Sheets files can be accessed from any device and can be converted to other formats, such as Excel or PDF.

LibreOffice Calc. This is a free and open source spreadsheet format, which is compatible with Excel and other formats. Calc files can be used to perform complex calculations, create pivot tables, and apply various formatting options.

OpenOffice Calc. This is another free and open source spreadsheet format, which is similar to LibreOffice Calc. OpenOffice Calc files can also be used to create and manipulate spreadsheets, and can be converted to other formats, such as Excel or PDF.

VisiCalc. This is one of the first spreadsheet formats, which was developed in the late 1970s. VisiCalc files were used to perform simple calculations and display data in rows and columns. VisiCalc files are no longer in use, but can be opened by some modern spreadsheet applications.
